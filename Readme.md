How to install this plugin

---

* After downloading, extract all plugin files to the /home/blesta/public\_html/plugins directory.
* Run the following command in your SSH terminal: sudo chown -R blesta:blesta /home/blesta/public\_html/plugins/virtfusion\_push/
* Install the plugin via the "Available Plugins" section in Blesta.
* Enjoy



## Features



* Client Access Control: Enable Push functionality for all clients or specific Client IDs.
* Package Control: Enable Push functionality for all packages or specific packages.
* Flexible Billing: Configure Push fees and the base currency. The system automatically generates invoices and converts the fee to the client's preferred currency using Blesta's built-in exchange rates.
* Cooldown Period: Set a mandatory cooldown period between transfers.
* Logs: View full activity logs for all Push operations.
